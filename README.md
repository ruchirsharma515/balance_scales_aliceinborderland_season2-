# balance_scales_aliceinborderland_season2-
So it is a fun Math game where you have to guess the numbers to win the game. I am not gonna reveal the surprise watch King of Diamonds game of Alice in Borderland Season 2 .

The game is based on the "Keynesian Beauty Contest" from economics. Theoretically, since the target number is always the average multiplied by 0.8, the logical deduction is that the "correct" guess will eventually converge down to 0. However, if everyone figures this out and guesses 0, no one wins, and the game stagnates.
